

==============================
MAGISK MODULE GUIDE
===================

BOOT ANIMATION:
system/media/bootanimation.zip

RINGTONES:
system/media/audio/ringtones/

NOTIFICATIONS:
system/media/audio/notifications/

ALARMS:
system/media/audio/alarms/

FONTS:
system/fonts/

SYSTEM APPS:
system/app/AppName/App.apk

PRIVILEGED APPS:
system/priv-app/AppName/App.apk

PRODUCT APPS:
system/product/app/

PRODUCT MEDIA:
system/product/media/

EXAMPLE:

system/
└── media
└── bootanimation.zip

After adding files:

zip -r ModuleName.zip ModuleName
